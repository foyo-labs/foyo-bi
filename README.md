# Foyo-BI

Foyo-BI 是一个面向业务、可视化、轻量的免费开源BI系统，目标是使用可视化工具完成数据图表与报表工作，适用于中小型企业应用。

### ***这个项目还属于早期开发阶段!***

功能:
1. **基于Web浏览器**: 在任何现代浏览器中运行
2. **易于使用**:
3. **图表设计器**:
4. **支持众多数据源**:
5. **可视化操作**:
6. **老板模式**:
7. **无需要解SQL语法**:


## 使用的关键技术

* [Kotlin](https://kotlinlang.org)
* [Spring](https://spring.io)
* [Ignite](http://ignite.apache.org)
* [Postgresql](https://www.postgresql.org)
